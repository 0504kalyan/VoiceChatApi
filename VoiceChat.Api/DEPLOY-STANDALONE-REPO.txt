    VoiceChat.Api.csproj

Monorepo (ChatbotAI)
--------------------
Use Api/VoiceChat.Api/render.yaml (paths from monorepo root).

VoiceChatApi GitHub repo
------------------------
1. From ChatbotAI run:
     powershell -File Api\VoiceChat.Api\sync-to-VoiceChatApi-clone.ps1 -VoiceChatApiRepoRoot "D:\path\to\VoiceChatApi"

2. Commit from repo root:
     git add VoiceChat.Api/Dockerfile VoiceChat.Api/.dockerignore render.yaml
     git push origin main

3. Render (either):
   - Blueprint: render.yaml at repo root (dockerfilePath: VoiceChat.Api/Dockerfile, dockerContext: VoiceChat.Api), OR
   - Manual: Root Directory = empty ; Dockerfile Path = VoiceChat.Api/Dockerfile ; Docker Context = VoiceChat.Api
