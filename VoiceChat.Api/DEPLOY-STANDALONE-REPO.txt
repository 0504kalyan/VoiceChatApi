Monorepo (ChatbotAI)
--------------------
Docker + Blueprint live at repository root:
  Dockerfile
  .dockerignore
  render.yaml

Paths inside Dockerfile: Api/VoiceChat.Api/

Blueprint path in Render: render.yaml (default) or leave empty for auto-detect.

VoiceChatApi clone
-------------------
  powershell -File Api\VoiceChat.Api\sync-to-VoiceChatApi-clone.ps1 -VoiceChatApiRepoRoot "D:\path\to\VoiceChatApi"

Then:
  git add Dockerfile render.yaml .dockerignore
  git push

Render: Root Directory = . ; Dockerfile Path = Dockerfile ; Docker Context = .

Runtime (required)
------------------
Set ConnectionStrings__DefaultConnection to a cloud SQL Server (e.g. Azure SQL). Local Server=. will crash on startup.
Jwt__SigningKey (>=32 chars), Cors__Origins__0 for production.
